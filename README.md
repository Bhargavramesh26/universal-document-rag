# Universal Document RAG Assistant

## Overview

An AI-powered Retrieval-Augmented Generation (RAG) application that enables users to upload documents and ask questions based on their contents using modern LLMs.

## Features

- Document upload
- Session management
- ChromaDB vector storage
- Semantic retrieval
- Multi-model support (Ollama, Gemini, Groq)
- React frontend
- FastAPI backend

## Tech Stack

### Backend
- FastAPI
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Ollama / Gemini / Groq

### Frontend
- React
- Vite
- Axios
- React Context API
